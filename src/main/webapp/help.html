This plugin understands the JSON Benchmark reports. This plug-in only displays useful information about results, such as average time, 90% percentile, median, historical result trend.

To use this plugin, first set up your build to run benchmarks. You can use tools such as PHPLame. Then you have to specify the path to the different benchmark reports, by default the plugin will use the **/*-REPORT.json pattern.

Example of the format: 
[
  {
    "min": "0",
    "median": "0",
    "title": "com.mongodb.performance.bson.Generate.generateObjectId",
    "max": "0.029",
    "executionTime": "19.123",
    "90percentile": "0",
    "invocations": 2000000.0,
    "average": "0.0000045715"
  }
]
